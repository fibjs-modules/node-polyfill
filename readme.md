## @fibs/node-polyfill

[![Build Status](https://travis-ci.org/fibjs-modules/mkdirp.svg)](https://travis-ci.org/fibjs-modules/mkdirp)

Polyfill for fibjs to be like Node.js

## Usage

```
  npm install @fibs/node-polyfill --save
```

## Usage

Just require it!

```js
  require('@fibs/node-polyfill')();
```
